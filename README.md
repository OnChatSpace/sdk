## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/onchat.git
cd onchat
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

## 🚀 Build

To build the project for production:

```bash
npm run build
```
